## 接口清单

|---|---|---|---|
| POST | `/novels/search` | 搜索可用于生成剧本的参考小说 | confirmed |

## 搜索小说

### 请求

`POST /novels/search`

| 名称 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| keyword | body | string | 是 | 小说名、作者或关键词 |

### 响应

```json
{
  "success": true,
  "data": [
    {
      "id": "novel-id",
      "title": "小说名",
      "author": "作者",
      "description": "摘要",
      "source": "来源"
    }
  ],
  "message": "可选说明"
}
```

### 错误码

当前接口没有稳定错误码；失败通过 `success: false` 与 `message` 表达。

### 联调说明

- `novelApiProvider` 为 `ai` 时，结果由 AI 生成。
- `novelApiProvider` 为 `compatible` 或 `custom-functions` 时，服务端调用配置的远程小说 API。
- `novelApiProvider` 为 `disabled` 时返回 `success: false`、空数组和配置提示。

### 协议偏差

- MISSING：没有字段级校验错误响应。
- MISSING：没有分页、结果总数或排序契约。
